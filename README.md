# Trabalho de Programação para Internet I 
## Portfolio em HTML e CSS puros
### Grupo Responsável Pelo Projeto
Os alunos que são responsáveis pelo projeto são: Isaac Borges Laranjeira, Paulo Gabriel e Leonardo José.

### Tema escolhido
O tema escolhido pelo grupo foi o portfólio para uma equipe de desenvolvimento Web composta pelos membros do grupo.

### Descrição do Portfólio
O projeto consiste no desenvolvimento de um portfólio digital para uma equipe focada em soluções de Engenharia Web. O objetivo principal do site é apresentar a equipe, destacar as competências individuais de cada desenvolvedor e exibir projetos para atrair potenciais clientes.

A arquitetura do projeto foi construída utilizando exclusivamente HTML5 e CSS3 puros.

O portfólio está dividido em quatro páginas principais:
- Home (index.html): Apresentação inicial da equipe, com um resumo dos membros e os links de navegação principais.

- Sobre Nós (sobre_nos.html): Uma página detalhada sobre o perfil de cada integrante.

- Currículo/Projetos (curriculo.html): Uma página que descreve projetos feitos pela equipe.

- Contato (nos_contate.html): Uma página contendo um formulário semântico e acessível.
### Principais Desafios
Os principais desafios foram encontrados em alguns momentos, sendo estes: a organização semântica do HTML, a organização do CSS e aplicar responsividade no projeto.

- Para organizar semânticamente e se libertar do uso incontrolável de divs foi um desafio pois era um hábito e a importância de usar tags
semânticas era desconhecida até então. 

- Organizar o CSS e aplicar ele no site foi complicado principalmente na hora de usar flex e grids, que já são conteúdos mais avançados do CSS.

- A responsividade foi tediosa no sentido que era necessário ver se o css foi bem aplicado em várias telas.
